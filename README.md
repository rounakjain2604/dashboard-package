# DCF Dashboard Package

## Run
1. Install dependencies:
   - `pip install -r requirements.txt`
2. Start dashboard:
   - `python dashboard_api.py`
   - or double-click `run_dashboard.bat`
3. Open:
   - `http://localhost:5050`

## Included
- Dashboard backend API (`dashboard_api.py`)
- Dashboard frontend (`templates/dashboard.html`)
- Core DCF engine (`src/dcf_engine/...`)
- Sample data (`data/asian_street_financials.csv`)
- Preset configs (`config.asian_street_ib_grade.json`, `config.example.json`)
